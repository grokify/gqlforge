# Release Notes: v0.1.0

**Release Date:** 2026-06-07

## Overview

gqlforge is a GraphQL client SDK generator for Go that orchestrates [genqlient](https://github.com/Khan/genqlient) to generate type-safe client code from GraphQL schemas and operations.

## Highlights

- **GraphQL Introspection** - Fetch schemas from any GraphQL endpoint
- **SDL Export** - Convert introspection results to Schema Definition Language
- **goauth Integration** - Flexible authentication via credentials file or bearer token
- **genqlient Workflow** - Scaffold and generate type-safe Go clients

## Features

### Introspection Package

- GraphQL introspection client with full schema query
- `ToSDL()` method for human-readable schema output
- `ToJSON()` method for raw introspection export
- Support for all GraphQL types: Object, Interface, Union, Enum, Input, Scalar

### CLI Commands

| Command | Description |
|---------|-------------|
| `introspect` | Fetch GraphQL schema via introspection query |
| `init` | Initialize genqlient project structure |
| `generate` | Generate Go client code using genqlient |
| `version` | Display version information |

### Authentication

Multiple authentication methods supported:

```bash
# goauth credentials file
gqlforge introspect --creds credentials.json --account myapi https://api.example.com/graphql

# Bearer token flag
gqlforge introspect --token YOUR_TOKEN https://api.example.com/graphql

# Environment variable
GQLFORGE_TOKEN=token gqlforge introspect https://api.example.com/graphql
```

## Workflow

```
GraphQL Endpoint
      ↓
gqlforge introspect
      ↓
schema.graphql
      +
operations/           ← You define these
  queries.graphql
      ↓
gqlforge generate (wraps genqlient)
      ↓
generated/client.go
```

## Installation

```bash
go install github.com/grokify/gqlforge/cmd/gqlforge@latest
```

## Quick Start

```bash
# Initialize project
gqlforge init myapi

# Introspect schema
cd myapi
gqlforge introspect --token YOUR_TOKEN https://api.example.com/graphql

# Define operations in operations/*.graphql
# Then generate client
gqlforge generate
```
