## [v0.2.0] - 2026-06-07 ([`10ebd97`](https://github.com/grokify/gqlforge/commit/10ebd97))

### Highlights

- Operation scaffolding from schema Query/Mutation types
- Schema validation and drift detection
- File watching with auto-regeneration
- Embedded genqlient (no CLI installation required)

### Added

- `scaffold` command to generate stub operations from schema types ([`8ce91db`](https://github.com/grokify/gqlforge/commit/8ce91db))
- Support --type, --depth, --include, --exclude flags for scaffold filtering ([`8ce91db`](https://github.com/grokify/gqlforge/commit/8ce91db))
- scaffold package with Generator for operation generation ([`8ce91db`](https://github.com/grokify/gqlforge/commit/8ce91db))
- `validate` command to check operations against schema ([`3e3eb81`](https://github.com/grokify/gqlforge/commit/3e3eb81))
- Support --strict mode and --json output for validation ([`3e3eb81`](https://github.com/grokify/gqlforge/commit/3e3eb81))
- `diff` command to compare local schema to remote endpoint ([`0c63281`](https://github.com/grokify/gqlforge/commit/0c63281))
- Detect added/removed types and fields with breaking change identification ([`0c63281`](https://github.com/grokify/gqlforge/commit/0c63281))
- Support --breaking-only and --json output for diff ([`0c63281`](https://github.com/grokify/gqlforge/commit/0c63281))
- `watch` command for auto-regeneration on file changes ([`bbd943b`](https://github.com/grokify/gqlforge/commit/bbd943b))
- Debounced file watching with fsnotify ([`bbd943b`](https://github.com/grokify/gqlforge/commit/bbd943b))

### Changed

- **BREAKING:** `generate` command now uses genqlient as embedded library instead of exec ([`303bb53`](https://github.com/grokify/gqlforge/commit/303bb53))
- genqlient CLI no longer required to be installed separately ([`303bb53`](https://github.com/grokify/gqlforge/commit/303bb53))

### Dependencies

- github.com/Khan/genqlient v0.8.1 (direct, embedded generation) ([`b387445`](https://github.com/grokify/gqlforge/commit/b387445))
- github.com/vektah/gqlparser/v2 v2.5.33 (direct, validation) ([`b387445`](https://github.com/grokify/gqlforge/commit/b387445))
